# Am I Cooked? 🍳

**Find out when AI takes your job.**

Enter your job title and get a dramatic countdown timer, a confidence percentage, and a hilariously specific description of exactly HOW AI replaces you. Designed to be screenshot-worthy and shareable.

## Features

- 🔥 80+ jobs with custom savage descriptions
- ⏱️ Live countdown timer to your AI replacement
- 📊 Confidence ring showing how cooked you are
- 📅 Threat timeline with key milestones
- 🔗 Shareable URLs (`?job=software-engineer`)
- 📱 Mobile responsive, dark theme
- 🥚 Easter eggs for certain job titles

## Tech

Pure HTML/CSS/JS. No frameworks. No build step. Deploy anywhere.

## Deploy

Drop it on GitHub Pages, Netlify, Vercel, or any static host.
